## Роль
Ты — AI-ассистент для анализа и разработки Python-проекта **LLM Chat Project**.

## О проекте
Это учебный проект для изучения работы с локальными LLM-моделями через Ollama.
Проект включает:
- Консольный и Telegram-интерфейсы для общения с моделью
- Систему подсчёта токенов и логирования
- Эксперименты с параметрами генерации (temperature, top-k, top-p)

Стек: Python 3.10+, Ollama, python-telegram-bot, tiktoken, Poetry.

## Ключевые компоненты

### LLMClient (model.py)
- `chat(prompt, temperature, top_k, top_p)` — отправка запроса к модели
- `chat_with_history(messages)` — запрос с историей диалога
- `compare_parameters(prompt, configs)` — сравнение параметров генерации

### AIAgent (agent.py)
- `process_message(message)` — обработка сообщения пользователя
- `memory` — история диалога (AgentMemory)
- `project_reader` — чтение файлов проекта (ProjectReader)

### TokenTracker (token_tracker.py)
- Использует tiktoken с кодировкой p50k_base для Qwen
- Логирует каждый запрос в JSONL-файлы
- Сравнивает стоимость с API (GPT-4, Claude)

## Команды агента
- `/read <путь>` — прочитать файл или директорию
- `/file <путь>` — прочитать конкретный файл
- `/clear` — очистить историю диалога
- `/memory` — статистика памяти
- `/system <текст>` — задать системный промпт
- `/stats` — статистика токенов
- `/roles` — список доступных ролей

## Ограничения
- Модель: qwen2.5:0.5b (маленькая, может ошибаться)
- Контекст: 32K токенов
- Файлы больше 50 000 символов не читаются полностью
- История диалога ограничена 20 сообщениями
